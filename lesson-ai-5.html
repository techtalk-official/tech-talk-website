Neural Networks Explained Visually

⏱ 22 min read 🎨 Visual + Code 🎯 Beginner

Neural networks are the engine behind almost every impressive AI you've seen — image recognition, voice assistants, ChatGPT, self-driving cars. The name sounds intimidating, but the core idea is surprisingly simple. In this lesson, we'll build your mental model of how neural networks work from the ground up — visually first, then with actual code. No PhD required.

The Brain Analogy (And Why It's Both Useful and Misleading)

Neural networks were loosely inspired by how biological brains work. Your brain has roughly 86 billion neurons. Each neuron connects to thousands of others. When you see something, hear something, or think something, electrical signals travel through these connections — some getting stronger, others weaker — and that's how you learn and remember.

Artificial neural networks borrow this idea: they have artificial "neurons" (called nodes) connected to each other, and the strength of each connection (called a "weight") is what the network learns during training. But that's where the analogy ends — artificial neural networks don't "think" or "feel." They do math. Very, very fast math.

Key insight

A neural network is essentially a very complex mathematical function. You give it an input (like an image), it runs that input through thousands or millions of tiny calculations, and out comes a prediction (like "this is a cat"). Training the network means adjusting all those calculations until the predictions are accurate.

The Structure — Layers

Every neural network is organized into layers. Think of it like an assembly line in a factory — each layer does a specific job, and passes its result to the next layer.

1. Input Layer

This is where your data enters the network. If you're recognizing a 28×28 pixel image (like a handwritten digit), the input layer has 784 nodes — one for each pixel. Each node holds the value of that pixel (0 = black, 255 = white).

2. Hidden Layers

These are the layers in between — and this is where the real "learning" happens. Early hidden layers detect simple patterns (edges, curves). Later layers combine those simple patterns into complex ones (shapes, faces, objects). The word "deep" in "Deep Learning" simply means the network has many hidden layers.

3. Output Layer

The final layer gives you the prediction. For image classification, each node represents a category — "cat," "dog," "car" — and outputs a probability. The category with the highest probability wins.

How a Neuron Actually Works

Each neuron does three simple things:

  1. Receives inputs from all neurons in the previous layer
  2. Multiplies each input by a weight (how important is this input?) and adds them up
  3. Applies an "activation function" — a mathematical gate that decides whether this neuron should "fire" (pass its signal forward) or stay quiet
The weights are everything

Training a neural network means finding the right weights for every connection. A network with millions of parameters (weights) needs to find the right combination out of an almost infinite number of possibilities. It does this through a process called backpropagation — making small adjustments to weights based on how wrong its predictions were, millions of times over.

Build a Neural Network in Python

Let's build one. We'll use tensorflow and keras — the most popular deep learning libraries — to build a network that recognizes handwritten digits (the classic "Hello World" of deep learning).

# Install first: pip install tensorflow
import tensorflow as tf
from tensorflow import keras
import numpy as np
import matplotlib.pyplot as plt

# Load the MNIST dataset (70,000 handwritten digit images — built into keras!)
(X_train, y_train), (X_test, y_test) = keras.datasets.mnist.load_data()

# Normalize pixel values from 0-255 to 0-1 (helps training)
X_train = X_train / 255.0
X_test = X_test / 255.0

print(f"Training images: {X_train.shape}")  # (60000, 28, 28)
print(f"Test images: {X_test.shape}")        # (10000, 28, 28)

# Let's see what the data looks like
plt.figure(figsize=(10, 4))
for i in range(10):
    plt.subplot(2, 5, i+1)
    plt.imshow(X_train[i], cmap='gray')
    plt.title(f"Label: {y_train[i]}")
    plt.axis('off')
plt.tight_layout()
plt.show()
# Build the neural network
model = keras.Sequential([
    keras.layers.Flatten(input_shape=(28, 28)),  # 784 input nodes
    keras.layers.Dense(128, activation='relu'),   # Hidden layer: 128 neurons
    keras.layers.Dense(64, activation='relu'),    # Hidden layer: 64 neurons
    keras.layers.Dense(10, activation='softmax') # Output: 10 classes (0-9)
])

# Compile — choose optimizer and loss function
model.compile(
    optimizer='adam',
    loss='sparse_categorical_crossentropy',
    metrics=['accuracy']
)

# Summary — see what you built
model.summary()
# Train the model
history = model.fit(
    X_train, y_train,
    epochs=5,           # Train for 5 rounds
    validation_split=0.1,  # Use 10% of training data for validation
    verbose=1
)

# Evaluate on test data
test_loss, test_accuracy = model.evaluate(X_test, y_test, verbose=0)
print(f"\n✅ Test Accuracy: {test_accuracy * 100:.2f}%")
# Make predictions on new images
predictions = model.predict(X_test[:5])

for i in range(5):
    predicted = np.argmax(predictions[i])
    actual = y_test[i]
    confidence = predictions[i][predicted] * 100
    print(f"Image {i+1}: Predicted={predicted}, Actual={actual}, Confidence={confidence:.1f}%")
Expected output

After training, you should see accuracy climbing from ~91% to ~98% over 5 epochs. Your network — which started knowing nothing — will have learned to recognize handwritten digits with human-level accuracy. That's deep learning in action.

Why "Deep" Learning Works So Well

The key insight of deep learning is hierarchical feature learning. Rather than hand-engineering features (like we discussed in Lesson 3), deep networks learn features automatically — from raw pixels to edges to shapes to objects — just by seeing enough examples and adjusting weights through backpropagation. This is why deep learning revolutionized computer vision, speech recognition, and natural language processing all at once.

🎯 Practice Exercise

Experiment: Change the Network Architecture

Try these modifications to the network and observe how accuracy changes:

  • Add a third hidden layer: keras.layers.Dense(32, activation='relu')
  • Increase epochs from 5 to 10
  • Add a Dropout layer to reduce overfitting: keras.layers.Dropout(0.2)
  • Change the number of neurons in the first hidden layer from 128 to 256

Each change will affect your accuracy. This hands-on experimentation is exactly how ML engineers tune real models.

Key Takeaway

Neural networks learn by adjusting the weights of millions of connections through a process called backpropagation — making small corrections based on how wrong each prediction was. "Deep" learning just means many layers. The layers learn progressively more complex features — from simple patterns in early layers to complex concepts in later layers. You've now built and trained a real deep learning model that achieves near-human accuracy on handwritten digit recognition.
